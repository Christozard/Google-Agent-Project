# System Architecture

## Tech Stack
- Frontend: Next.js 14, React 18, Tailwind CSS
- State Management: Zustand
- API: Node.js 20, Express
- Database: SQLite (development), PostgreSQL (production)
- MCP Integration: Moomoo API MCP v2.3

## Folder Structure
```
/src
  /agents
    /development
    /runtime
  /shared_memory
  /frontend
    /pages
    /components
  /api
```

## Key Design Decisions
1. Mono-repo structure for agent coordination
2. REST API for agent communication
3. File-based memory system

# Product Requirements

## Overview
Minimum Viable Product for Agentic Investment Copilot

## MVP Features
1. Basic portfolio analysis using Moomoo MCP
2. Investment Committee decision framework
3. Trade approval workflow with human confirmation
4. Core agent communication system

## User Stories
- As an investor, I want to see portfolio risk analysis
- As a trader, I need manual confirmation before execution
- As a developer, I require clear agent activity logging
